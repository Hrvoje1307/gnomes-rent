## 📌 Opis projekta

**Gnomes Rent** je web aplikacija za upravljanje najmom razvijena u čistom PHP-u.  
Projekt je jednostavan za pokretanje i ne zahtijeva dodatne frameworke ni kompliciranu konfiguraciju.

Ovaj repozitorij može služiti kao:
- studentski projekt
- baza za nadogradnju (npr. baza podataka, autentifikacija)
- portfolio primjer jednostavne PHP aplikacije

---

## 🚀 Pokretanje projekta lokalno

### 1️⃣ Kloniraj repozitorij

```bash
git clone https://github.com/Hrvoje1307/gnomes-rent.git
```

### 2️⃣ Uđi u direktorij projekta

```bash
cd gnomes-rent
```

### 3️⃣ Pokreni PHP development server

```bash
php -S 127.0.0.1:8000
```

### 4️⃣ Otvori aplikaciju u pregledniku

```
http://127.0.0.1:8000
```

## 🍴 Kako forkati projekt

Ako želiš napraviti svoju verziju projekta:

1. Klikni **Fork** gumb u gornjem desnom kutu GitHub stranice.
2. Nakon što se fork kreira na tvom profilu, kloniraj svoj fork:

```bash
git clone https://github.com/TVOJ_USERNAME/gnomes-rent.git
```

3. Napravi izmjene i pošalji ih:

```bash
git add .
git commit -m "Opis promjena"
git push origin main
```

4. Otvori **Pull Request** ako želiš predložiti izmjene originalnom projektu.
